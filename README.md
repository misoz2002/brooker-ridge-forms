# Brooker Ridge Forms

Custom WordPress forms for Brooker Ridge Animal Hospital.

Features include a detailed dashboard form editor, conditional logic, secure appointment and registration submissions, private WordPress submission backup, CSV export, file attachments, and optional Google Sheets synchronization.

The repository contains no patient submissions, passwords, database credentials, or Google shared secret.

## Updates

GitHub Actions packages each tag as an immutable `brooker-ridge-forms-X.Y.Z.zip` asset. Installed copies check the latest public GitHub release and display normal WordPress plugin updates.

Releases also include an identical `brooker-ridge-forms.zip` compatibility asset so installations using the pre-2.0.4 updater can migrate once; current versions select only the versioned asset.
